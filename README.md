# Shaqir Beqiri Portfolio

Modern portfolio built with **Node.js**, **Express**, **EJS**, and **Bootstrap**.

## Live Demo

- www.

## Tech Stack

- Node.js
- Express.js
- EJS
- Bootstrap 5
- Custom CSS

## Features

- Responsive modern UI
- Projects section with GitHub links
- About section with CV integration
- Contact section with icon-based links (Email, LinkedIn, GitHub)
- Contact form with server-side validation

## Project Structure

```bash
.
├── app.js
├── data/
│   ├── profile.js
│   └── projects.js
├── public/
│   ├── css/custom.css
│   └── docs/cv-shaqir-2026.pdf
├── routes/mainRoutes.js
└── views/
    ├── index.ejs
    ├── about.ejs
    ├── projects.ejs
    ├── 404.ejs
    └── partials/
```

## Run Locally

```bash
npm install
npm start
```

Open: `http://localhost:3000`
